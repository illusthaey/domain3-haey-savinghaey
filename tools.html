업무 효율화 도구 모음

실제 변환 도구는 별도 페이지로 확장하고, 여기서는 업무 링크/절차를 표준화합니다.

운영 원칙(권장)

1) 입력 → 변환 → 출력
반복업무는 단계 고정이 오류율을 줄입니다.
2) 데이터는 한 곳(store.json)에서만 관리
기기/브라우저 달라도 동일 결과를 보장합니다.
3) 기능 추가 시 페이지 단위 분리
코드 파편화를 막고 유지보수를 단순화합니다.

도구(예시)

PDF → 엑셀 변환
양식별 추출 규칙을 고정해 자동화
세입/징수 자료 정형화
현황 PDF를 표준 템플릿으로 변환
연말정산/기타소득 체크리스트
누락 방지: 체크리스트 + 증빙 링크 관리
계약/검수/대금지급 절차 메모
자주 틀리는 포인트를 규정처럼 저장
